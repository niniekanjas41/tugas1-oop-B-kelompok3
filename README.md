## 📖 Deskripsi
Program ini merupakan aplikasi sederhana berbasis Java yang berjalan di terminal (console-based) dan digunakan untuk melakukan simulasi reservasi hotel.

---

## 👤 Pengguna dapat:
- Melihat daftar kamar
- Melakukan reservasi
- Menghitung total biaya berdasarkan jumlah malam
- Melakukan proses check-in dan check-out

---

## 🧠 Konsep yang Digunakan
Program ini menerapkan konsep dasar Object-Oriented Programming (OOP):
- Class dan Object
- Constructor
- Encapsulation
- Method
